KAZ OS

Простая операционная система на базе Cosmos (C#)

Команды

Команда Описание
help Показать список команд
shutdown Выключение системы
reboot Перезагрузка
sysinfo Информация о системе (CPU, RAM)

Информация о системе

Команда sysinfo показывает:

· Модель процессора (CPU)
· Производителя CPU
· Общий объем ОЗУ
· Доступную память
· Используемую память

Установка

1. Установите Visual Studio 2022
2. Установите CosOS (Cosmos) из Visual Studio Marketplace
3. Создайте новый проект Cosmos
4. Замените код в Kernel.cs на мой
5. Соберите и запустите проект

Требования

· Visual Studio 2022
· Cosmos User Kit
